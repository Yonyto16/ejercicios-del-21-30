Etiqueta BLOCKQUOTE

El elemento blockquote permite a los autores insertar citas en forma de bloques de contenido, usualmente compuestos por un parrafo ,un grupo de parrafo o un conjunto de muchos otros elementos incluyendo imagens (img), tablas (table) y articulo (article), entre otros. Esta es la principal diferencia entre este elemento y q, que esta diseñado para citar unicamentes lines de texto.
