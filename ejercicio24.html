Etiqueta Section

un documento web podria dividirce en sectiones para un listado de notcias, informacion de contactos, etc.

Debemos asegurarno que todo lo que se encuentra dentro de una etiqueta SECTION tenga relación.

se usa para crear contenido independientes dentro de un documento web.

Etiqueta ARTICLE

un documeto web podria contener un numero indeterminado de articulos, ya que cada uno posee. independencia frente a los demas arcticulos inclucive dentro del mismo web.

en un articulo podria colocar etiquetas como HEADER, SECTION Y FOOTER.

La diferencia entre ARTICLE y SECTION es que un ARTICLE tiene una cultura semantica mas fuerte que un SECTION.
